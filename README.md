# Proyecto Final: Aplicación Web de Inventario

Esta aplicación es un sistema de control de inventario desarrollado con **PHP** y **MySQL**, integrando un diseño profesional con **Bootstrap** y validaciones en **JavaScript**.

## Funcionalidades
- **CRUD Completo:** Registro, visualización y eliminación de productos en tiempo real.
- **Base de Datos:** Gestión persistente de productos mediante MySQL.
- **Diseño Responsivo:** Interfaz moderna adaptada a dispositivos móviles.
